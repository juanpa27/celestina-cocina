# Dominio para Celestina Cocina

## Lo que ya tenés (gratis)

`celestina-cocina.vercel.app` — funciona ahora mismo, gratis para siempre.

---

## Recomendación — Dominio temático de cocina ✅

Son dominios con extensiones relacionadas a gastronomía. Se ven mucho mejor que `.com` para un negocio de comida.

| Dominio sugerido | Precio aprox./año | Por qué |
|---|---|---|
| **`celestina.kitchen`** | ~$28 USD | Corto, directo, on-brand ⭐ |
| `celestinacocina.kitchen` | ~$28 USD | Más completo |
| `celestina.cafe` | ~$22 USD | Cálido, funciona para comida casera |
| `celestina.menu` | ~$32 USD | Literal — es un menú digital |
| `celestina.recipes` | ~$38 USD | Para el futuro si sumás recetas |

> Menos de 1.500 Gs por día para el `.kitchen`. Vale la pena para el negocio.

### Dónde comprarlo

- **[Porkbun.com](https://porkbun.com)** — el más barato, interfaz muy simple, incluye privacidad WHOIS gratis
- **[Namecheap.com](https://namecheap.com)** — alternativa clásica, igual de confiable
- **[Cloudflare Registrar](https://cloudflare.com/products/registrar/)** — precio de costo sin markup (requiere cuenta)

Los tres son confiables y se conectan a Vercel en 5 minutos.

---

## Cómo conectarlo a Vercel (5 minutos)

### Paso 1 — En Vercel
```
Vercel → Tu proyecto → Settings → Domains
→ Escribís: celestina.kitchen  →  [Add]
→ Vercel te muestra los registros DNS:
    A      →  76.76.21.21
    CNAME  →  cname.vercel-dns.com
```

### Paso 2 — En Porkbun / Namecheap
```
Panel del registrador → DNS Management → Add Records:
    Tipo A     | Host: @   | Value: 76.76.21.21
    Tipo CNAME | Host: www | Value: cname.vercel-dns.com
```

### Paso 3 — Esperar
- DNS propaga en 5 a 30 minutos
- HTTPS/SSL lo activa Vercel automáticamente, sin costo extra
- Vercel muestra ✓ verde cuando está activo

## Resumen

Para Celestina Cocina, **`celestina.kitchen`** es la mejor opción:
- Corto y fácil de dictar por teléfono o WhatsApp
- Relacionado directamente con el negocio
- ~$28 USD/año en Porkbun
- Conexión a Vercel en 5 minutos
